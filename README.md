## üé® How to Use

Once installed, follow these simple steps to use the plugin:

1. **Open GrandMA3:** Start the GrandMA3 lighting console on your computer.
2. **Access the Plugin Menu:** Go to the plugins section in the main menu.
3. **Select UI-Color-List-Reference:** Click on the plugin to launch it.
4. **Browse Colors:** Use the interface to view available colors along with their reference strings.
5. **Choose Colors for Your UI:** Select colors that suit your design needs and easily copy their reference strings.
